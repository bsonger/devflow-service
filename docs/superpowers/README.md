# Superpowers

## Purpose

This directory holds process artifacts that are useful during design and implementation work but are not the active source of repository execution truth.

Use this directory for:

- design specs
- implementation plans
- TODO backlogs tied to an upcoming slice of work

Do not use this directory for:

- current service ownership
- current runtime behavior
- current API truth after implementation lands

Those facts belong in the main docs layers after the code exists.
