# Services

This directory contains current service and domain-facing docs for `devflow-service`.

Use:
- `meta-service.md` for the active service boundary and migration state
- `application.md` for the `Application` resource contract
- `project.md` for the `Project` resource contract
- `cluster.md` for the `Cluster` resource contract
- `environment.md` for the `Environment` resource contract
- `app-service.md`, `config-service.md`, `network-service.md`, `platform-orchestrator.md`, `platform-web.md`, `release-service.md`, `runtime-service.md`, and `service-common.md` for migrated cross-repo service summaries from `devflow-control`

These docs should describe the current code in this repo.
Do not treat migrated material from sibling repos as authoritative if it conflicts with the current implementation here.
